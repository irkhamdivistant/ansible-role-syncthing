# systemd role collection

This Ansible Collection contains a role for managing custom systemd unit files.

## Roles

- `systemd_config`: Deploy custom systemd unit templates.

## Requirements

- Ansible 2.9+
- Tested on Ubuntu, Debian, ArchLinux
